{Scatter plots}
  • The plot() command is used to draw a simple scatter plot of two vectors of the length.
  • Vectors x and y are defined in 'week9a.r'. 
    
>plot(x,y)
{Axes options: Labels}
  • Many plotting functions, such as plot(), allow you to include axis and text options.
  • We can use the parameters `xlab' and 'ylab' to label the x-axis and y-axis respectively. 

	
	>x=rnorm(15)  # generate 10 random numbers 
	>y=rnorm(15)  # using rnorm() for x and y
	>
	>plot(x,y)    # without labels
	>plot(x,y, xlab='x axis name', 
	+ ylab='y axis name')
{Axes options: Limits}
  • The plot() function will automatically fit the size of the plot to the data.
  • Should a larger plot area be required, the limits of the x-axis and y-axis can be specified either separately or in conjuction.
  • Use the range() function to determine suitable limits.
  • Specify the lower and upper limits as a vector.
{Axes options: Limits} 

	
	>plot(x,y)  # plot limits fitted to data
	>
	>
	> range(x)  
	[1] -1.687220  1.410094
	>
	> range(y)
	[1] -2.729680  3.195901
	>
	>plot(x,y, xlim=c(-2,2), ylim=c(-3.4,3.4))
{Text options: Plot text}
  • Text can be added to graphs using the text( ) function.
  • text( ) places text within the graph.
  • The first argument is the x and y co-ordinates of where to place the text.
  • Then the text to be used, in quotation marks.
  • Use location (x,y)=(-2.5,-3) for this example. 

	>text(-2.5,-3, "text to be added")
{Text options: Margin text}
  • Text can also be added to graphs using the mtext( ) function.
  • mtext( ) places text in one of the four margins.
  • First specify the text to be added.
  • Then select the margin to be used: (1=bottom, 2=left, 3=top, 4=right).
  • Character size and colour can be chosen using the `cex' and `col' parameters. 

	>mtext("margin text to be added ", side=4)
{Multiple plots} Multiple plots in a single graphic window
  • You can have more than one plot in a graphic window.
  • For example, par(mfrow=c(1,2))allows you to have two plots side by side.
  • par(mfrow=c(2,3)) allows 6 plots to appear on a page (2 rows of 3 plots each).
  • Note that the arrangement remains in effect until you change it.
  • If you want to go back to the one plot per page setting, type par(mfrow=c(1,1)).
  • This can be used for any combination of plots, i.e. boxplots, histograms, scatterplots etc.
{Multiple plots: Code} 

	
	par(mfrow=c(1,2)) 
	# two plots side by side
	
	boxplot(xdat,
	main='boxplot name',
	ylab='label of y-axis',
	xlab='label of x-axis')
	
	boxplot(xdat,
	main='boxplot name',
	xlab='label of x-axis',
	horizontal=TRUE)
	
	par(mfrow=c(1,1)) # reset
{Combining scatterplots}
  • It is possible to plot two sets of bivariate data on the same scatter plot.
  • This is done by adding a second set of data points to an existing scatterplot, using the points() command.
  • Graphical settings such as plot character and colour can be used to distinguish between both sets of data points.
  • Axes limits must be set so that all sets of observations are fully included in the plot.
  • Titles, plot text, etc can be added accordingly.
{Combining scatterplots: code} 

	
	> z=rnorm(15)
	> #x and y from before
	
	> range(z)
	[1] -1.527107  1.782608
	
	> plot(x,y, pch =18, col="red",
	xlim=c(-3.5,3.5),ylim=c(-3.5,3.5))
	> points(x,z, pch = 17, col="blue")
	> points(y,z, pch = 16, col="green")
{Adding legends}
  • A legend can be added to a plot using the legend() function.
  • The first parameter is the location of the legend box.
  • The location can be specified by using either the x-y co-ordinates or by specifying a location in the plot such as ``top", ``topright",`` bottomleft".
  • The legend parameter uses a character vector of the description of each set of points.
  • We use the `pch' and `col' parameters to specify the plot character and colour for each element of the legend. A vector is used in each case.
  • Obviously there should be a correspondence to the plot characters used in the plot.
{Adding legends: Code} 

	>legend("topright", 
	legend=c("x-y pts","x-z pts ","y-z pts"), 
	pch=c(18,17,16), 
	col=c("red","blue","green"))
