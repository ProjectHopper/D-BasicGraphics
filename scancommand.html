{5.2 The scan() command

  • The \texttt{scan()} function is a useful method of inputting data quickly.
  • You can use to quickly copy and paste values into the \texttt{R} environment. It is best used in the manner as described in the following example.
  • Create a variable ”X” and use the \texttt{scan()} function to populate it with values.
  • Type in a value, and then press return. Once you have entered all the values, press return again to return to normal operation.
