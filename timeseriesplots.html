Time Series Data

Time Series Analysis is not part of the course. However, plotting time series data is very simple. There are several plot types that can be used. To specify which type, use the \texttt{type=".."} argument in the \texttt{plot} command. \begin{description} \item[l] Line plot \item[p] Points plot \item[b] Line points with points superimposed (``b" is for both). \end{description} \begin{framed} \begin{verbatim} plot(Nile,type="l",col="red") \end{verbatim} \end{framed} \begin{center} \includegraphics[scale=0.55]{lineNile} \end{center}
