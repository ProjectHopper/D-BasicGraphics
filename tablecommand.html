Frequency Tables

  • A frequency distribution depicts the grouping of data divided into mutually exclusive categories and the frequency, i.e. number of occurrences in a category.
  • The R command used to construct frequency table is simply table()
  • The command can be used for one-way and two-frequency tables.


Frequency Tables

  • The \textit{\textbf{mtcars}} contains a binary variable for the two categories of transmission: am
  • 0 means automatic and 1 means manual.
  • How many cars in the data set are ``automatic" and how many are ``manual"? 

> table(mtcars$am)

 0  1 
19 13


  • There is a second categorical variables in the data set cyl.
  • Suppose we want construct a two-way frequency table for am and cyl .
  • Simply specify both as arguments to the table() command. 

> attach(mtcars)
> 
> table(am,cyl)
   cyl
am   4  6  8
  0  3  4 12
  1  8  3  2
